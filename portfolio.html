Sweet-Tooth

Sweet-Tooth

Sweet tooth is an in-development blood sugar tracker that utilizes Dexcom's proprietary equipment. The user can select from a list of provided virtual users to monitor someone's metrics in real time. The user can also provide their own blood sugar values in a fully dynamic d3 graph.

Weather The Storm

Weather the Storm

Weather the Storm provides the user to retrieve a weather forecast for any city in the United States. The last seven selections are shown to the user. The user is presented with the current conditions and a seven day forecast.

The Developer Brain Buster

The Brain Buster

The brain buster is a multiple choice quiz that allows the user to practice their developer knowledge. The user has a finite amount of time to complete the questions. There is a five second penalty for every wrong submission. Score is tallied for every answer.

Work Day Scheduler

Work Day Scheduler

The work day scheduler provides the user with a work day planner. The input is saved when the user clicks anywhere other than the text box. Past time is grey, current time is highlighted red, and future times are in green.
