Sweet-Tooth

Weather the Storm

The Brain Buster

The brain buster is a multiple choice quiz that allows the user to practice their developer knowledge. The user has a finite amount of time to complete the questions. There is a five second penalty for every wrong submission. Score is tallied for every answer.

Team Profiles


Team Profiles

Team Profiles, through Node, allows a user to create an employee profile database. The user inputs( node index.js ) in the terminal to start the program. The user can choose to create a profile card for Employee, Intern, Engineer, or Manager. The user can input info such as office number, email, github, etc. based off of the employee category selected. Team Profiles utilizes JSON data and adds to the existing organized creations. Once the the program is finished, all of the data is collected and formatted to an HTML( index. html ). The profiles are displayed as cards and are labeled based on employee position.

WeebHub


WeebHub

WeebHub is a fantastic website built using existing libraries and technologies such as: Handlebars, Passport, Sequelize, MySQL, Cookie-Parser and others. A user can create an account and login to read or leave reviews of their favorite animated show. The user must create an account to access the features WeebHub has to offer. Once the user is authenticated, the user is free to search through the database and read or leave a review.
